<a href="http://www.folkestonegerald.com/v/4/Chambers">Chambers</a> in assocation with <a href="http://www.folkestonegerald.com/v/1988/The%20Ship%20Inn">The Ship Inn</a> present their <a href="/wiki/easter+ales+festival">Easter Ales festival</a> this weekend! And I'm going to miss it, DOH! It's on from Thursday 24th - Monday 28th March, and the runners and riders are as follows... Your four regular ales are <a href="/wiki/adnams+southwold">Adnams Southwold</a>, <a href="/wiki/ramsgate+dark+ale">Ramsgate Dark Ale</a>, <a href="/wiki/hopdaemon+skrimshander">Hopdaemon Skrimshander</a> and <a href="/wiki/ringwood+old+thumper">Ringwood Old Thumper</a>. These beers are generally to be found at the bar in Chambers anyway. There are 12 guests lined up, these may change but right now they're <a href="/wiki/church+end+bats+in+the+belfry">Church End Bats in the Belfry</a>, <a href="/wiki/titanic+iceberg">Titanic Iceberg</a>, <a href="/wiki/black+sheep+best">Black Sheep Best</a>, <a href="/wiki/hanby+cherrybomb">Hanby Cherrybomb</a>, <a href="/wiki/trip+fff+comfortably+numb">Trip FFF Comfortably Numb</a>, <a href="/wiki/widdey+valley+golden+rivet">Widdey Valley Golden Rivet</a>, <a href="/wiki/wye+valley+owd+sarah">Wye Valley Owd Sarah</a>, <a href="/wiki/ramsgate+dogbolter">Ramsgate Dogbolter</a>, <a href="/wiki/spring+head+roaring+meg">Spring Head Roaring Meg</a>, <a href="/wiki/downton+brewing+co+chimera+gold">Downton Brewing Co Chimera Gold</a>, <a href="/wiki/hook+norton+double+stout">Hook Norton Double Stout</a>, and <a href="/wiki/on+the+good+ship+chambers+festival+ale">On The Good Ship Chambers Festival Ale</a> brewed especially for the festival by Hopdaemon! The four guest ciders are all rippingly strong; <a href="/wiki/biddendens+dry">Biddendens Dry</a>, <a href="/wiki/inch+s+west+country+dry">Inch's West Country Dry</a>, <a href="/wiki/thatchers+cheddar+valley">Thatchers Cheddar Valley</a>, and <a href="/wiki/winkleigh+scrumpy+med+dry">Winkleigh Scrumpy Med Dry</a>...

Enjoy!
