There's another beer festival at <a href="/wiki/chambers">Chambers</a> over the bank holiday weekend, ace! The last one in May was a roaring success, don't miss out. Plenty of cider on tap too, get your drinking trousers ready... live bands on each night including <a href="/wiki/sick+note+blues">Sick Note Blues</a> on the Friday. But I think we'll still go anyway.

The brews lined up so far are <a href="/wiki/wight+spirit">Wight Spirit</a>, <a href="/wiki/salopian+lemon+dream">Salopian Lemon Dream</a>, <a href="/wiki/black+zac">Black Zac</a>, <a href="/wiki/cherry+bomb">Cherry Bomb</a>, <a href="/wiki/radgie+gadgie">Radgie Gadgie</a>, <a href="/wiki/chancellor+s+revenge">Chancellor's Revenge</a>, <a href="/wiki/dragonhead+stout">Dragonhead Stout</a>, <a href="/wiki/skrimshander">Skrimshander</a>, <a href="/wiki/goachers+best+dark+ale">Goachers Best Dark Ale</a>, <a href="/wiki/gadd+s+no+7">Gadd's No. 7</a>, <a href="/wiki/village+bitter">Village Bitter</a>, and <a href="/wiki/49er">49er</a>. The four ciders are <a href="/wiki/snake+catcher">Snake Catcher</a>, <a href="/wiki/cheddar+valley">Cheddar Valley</a>, <a href="/wiki/old+rosie">Old Rosie</a> and the death by apples <a href="/wiki/biddenden+s+dry">Biddenden's Dry</a> at 8.0%.
